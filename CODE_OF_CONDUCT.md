# 行为准则

<p align="center">
  <strong>中文</strong> | <a href="CODE_OF_CONDUCT.en.md">English</a>
</p>

## 我们的承诺

为了营造开放、友善、包容的社区环境，我们作为贡献者与维护者承诺：无论年龄、体型、身体健全与否、民族、性别认同与表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教、性取向，**人人参与本项目与社区时均不受骚扰**。

## 我们的标准

有助于创造积极环境的行为包括：

- 使用友善、包容的语言
- 尊重不同的观点与经验
- 耐心、建设性地接受批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心

不可接受的行为包括：

- 使用性化的语言或图像，以及不受欢迎的性关注或挑逗
- 挑衅、侮辱/贬损性评论，以及人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可，发布他人的私人信息（如住址、电子邮箱）
- 其他在专业场合可被合理认定为不适当的行为

## 我们的责任

项目维护者有责任澄清可接受行为的标准，并应对任何不可接受行为采取适当、公平的纠正措施。

维护者有权且无义务删除、编辑或拒绝与本行为准则不符的评论、提交、代码、Wiki 编辑、Issue 及其他贡献；可暂时或永久禁止任何贡献者从事上述行为。

## 适用范围

本行为准则适用于项目空间及代表项目或社区公开活动的场合。代表项目或社区的情形包括但不限于：使用官方项目邮箱、通过官方社交媒体账号发布内容、作为指定代表出席线上或线下活动。项目的具体代表可由维护者进一步界定。

本准则亦适用于项目相关的 **GitHub Issues、Pull Request、Discussions**，以及 README 中列出的 **QQ 群、微信公众号** 等交流渠道。

## 执行

若你遭遇或目睹滥用、骚扰或其他不可接受行为，请通过以下方式联系项目团队：

- **电子邮箱**：[feiyangtech@qq.com](mailto:feiyangtech@qq.com)
- **GitHub**：向维护者私信，或在相关 Issue/PR 中 `@` 维护者（请勿在公开处披露他人隐私信息）

所有投诉将被及时、公正地审查与调查。项目团队有义务对举报人保密。对违规者的处理可能包括警告、临时禁言或永久移出社区，具体由维护者酌情决定。

不遵守或恶意违反本行为准则者，可能面临项目维护者、组织代表或社区成员依据项目治理规则采取的进一步措施。

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org) 2.1 版，原文见：<https://www.contributor-covenant.org/version/2/1/code_of_conduct.html>。

社区影响指南参见：<https://www.contributor-covenant.org/version/2/1/code_of_conduct/#community-impact-guidelines>。
