核心优势

写得顺、找得快、总结省力

不是替代 Word 或 Notion 的通用编辑器，而是面向持续记录、回顾与汇报的工作日志伴侣。

数据在本地

日志、记忆与设置落盘在你指定的目录，不绑定云端笔记平台。

找得快

关键词 + 语义检索，用一句话找回几个月前的细节与决策背景。

总结省力

勾选多篇日志，一键生成日报、周报、邮件与阶段汇报草稿。

模型自配

大语言、多模态与嵌入模型均由你在设置中接入，能力由你掌控。

产品界面

左侧组织日志，右侧沉浸写作

树形管理、富文本编辑、本地搜索与工作台 AI 辅助，在同一桌面窗口中连贯完成。

主要功能

从记录到检索，从总结到备份

日志组织与编辑

  • 富文本：标题、列表、任务、表格、代码块、公式与多媒体
  • 批量移动 / 删除，拖拽整理，Markdown 导入
  • 安装版：智能补全、图片粘贴与拖拽（本地学习，不上传正文）

搜索与理解

  • 本地关键词搜索，多处命中片段一键跳转
  • 配置嵌入模型后支持语义检索，未配置时自动降级

工作台

  • 记忆：跨日志的约定与术语，供总结与问答参考
  • 日志总结：日报 / 周报 / 月报 / 邮件 / 项目报告等
  • 日志问答：检索相关片段归纳作答，标注参考来源

设置与数据

  • 浅色 / 深色主题，中 / 英文，界面缩放可调
  • 自定义日志目录，.ws 导出与合并恢复
  • 分别配置 LLM、多模态与嵌入模型，支持连接测试

安装版 vs 开发版

快速上手

四步开始你的本地工作日志

  1. 01

    下载安装

    从 Releases 获取安装版或开发版安装包。

  2. 02

    完成设置

    选择主题、语言、日志目录；按需配置模型并测试连接。

  3. 03

    持续记录

    左侧组织日志，右侧富文本写作，Ctrl+S 保存并更新索引。

  4. 04

    回顾与汇报

    搜索找回细节，在工作台总结、问答或导出 .ws 备份。
